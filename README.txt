==================================================
Django - Simple Gravatar (version 0.1)
http://www.tomatohater.com/django-simple-gravatar/
==================================================

A Django plugin application to add Gravatar support 
to your Django projects.

To install this app, simply:
1) Drop the 'simplegravatar' folder somewhere onyour PYTHONPATH. 
2) Add 'simplegravatar' to your projects INSTALLED_APPS list in your settings.py file.
3) Add the path to 'PYTHONPATH/simplegravatar/templates' to TEMPLATE_DIRS in your settings.py file.

Enjoy!
